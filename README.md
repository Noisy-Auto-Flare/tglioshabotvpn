# 🚀 Telegram VPN Subscription Bot (@StingerVPN Edition)

Полнофункциональный Telegram-бот для продажи подписок на VPN. Проект оптимизирован для работы на слабых VPS (от 1 ГБ ОЗУ) и включает в себя автоматическую оплату через множество шлюзов, управление пользователями в панели RemnaWave и продвинутую систему лояльности.

## ⚙️ Настройка окружения (.env)

Создайте файл `.env` на основе `.env.example`. Ниже приведено подробное описание ключевых переменных:

### 🤖 Telegram Бот
- `BOT_TOKEN`: Токен вашего бота от [@BotFather](https://t.me/BotFather).
- `ADMIN_IDS`: ID администраторов через запятую (дают доступ к рассылкам и обход проверки подписки).
- `REQUIRED_CHANNEL`: Username канала для обязательной подписки (например, `@StingerVPN`).

### 🔑 VPN Панель (RemnaWave)
- `REMNAWAVE_API_URL`: URL вашей панели (например, `https://vpn.example.com/api`).
- `REMNAWAVE_API_KEY`: API ключ (Настройки -> API Keys).
- `REMNAWAVE_COOKIE`: Cookie авторизованной сессии.
- `SUB_DOMAIN`: Домен для ссылок подписки (например, `https://sub.example.com`).
- `REMNAWAVE_DEFAULT_SQUAD_UUID`: ID группы (Squad), в которую будут добавляться новые пользователи.

### 💳 Платежные шлюзы
Поддерживаются Platega (СБП), CryptoBot, CryptoMus, Telegram Stars и TON Connect. Настройка каждого шлюза описана в `.env.example`. 

**Важно для Platega:**
Для автоматического подтверждения платежей через Platega, укажите в личном кабинете Platega следующий Callback URL:
`https://ваш-домен.com/api/v1/payments/platega/webhook`

## 🚀 Быстрый старт (Docker)

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/your-username/tg-vpn-bot.git
   cd tg-vpn-bot
   ```

2. **Настройте конфиг**:
   ```bash
   cp .env.example .env
   # Отредактируйте .env, указав ваши ключи и настройки
   nano .env
   ```

3. **Запустите проект**:
   ```bash
   docker-compose up -d --build
   ```

## �️ Команды и управление

### Для пользователей:
- `/start` — Запуск бота, проверка подписки и регистрация (в т.ч. реферальная).
- **Профиль** — Просмотр ID, баланса, статуса подписки и ключа.
- **Инструкции** — Получение ссылки на настройку (динамически).

### Для администраторов:
- **Рассылка**: Просто отправьте любое сообщение боту (текст, фото или видео), и он предложит разослать его всем пользователям.
- `/broadcast [текст]` — Быстрая текстовая рассылка.
